HTML Elements

Below is a table of 41 commonly used HTML elements.

Tag Name Description Block,Inline or N/A Container or Empty
<!-- --> HTML commennts are invisible on browser. It is used to explain code. N/A Container
<!DOCTYPE> It is an instruction to the web browser telling it which version of HTML to use. N/A Empty
<a> Defines a hyperlink. Linking it to different pages. Inline Container
<article> It specifies independent, self contained content. Block Container
<aside> It defines some content other than the content it is placed it.The aside content should nbe related to the surrounding content. Block Container
<body> It defines the body of the document. N/A Container
<br/> It inserts a single line break. Inline Empty
<dd> It describes a term or name in a description list. Block Container
<div> It defines a section in an HTML doc. Block Container
<dl> It defines a description list. Block Container
<dt> It defines a term or name in a description list. Block Container
<em> It renders as emphasized text. Inline Empty
<fieldset> It is used to group related elements in a form. Block Container
<figcaption> It defines a caption element. Block Empty
<figure> It specifies self-contained content. For example illustrations, photos, diagrams. Block Container
<footer> It defines a footer for a document or section Block Container
<form> It is used to create an HTML form for user input Block Container
<h1>to<h6> They are used to define HTML Headings. Block Container
<head> It is a container for all the head elements. Example title, scripts, styles. Block Container
<header> It represents a container for introductory content or set of navigational links. Block Container
<html> It tells the browser that this is an HTML document. Block Container
<iframe> It is used to embed a document within the currewnt HTML document. Inline Container
<img/> Defines an image in an HTML document. Inline Empty
<input> Specifies an input field where the user can enter data. Inline Empty
<label> Defines a label for a button, input, output etc. Inline Container
<li> Defines a list item. Block Container
<link/> Defines a link between a document and an external document. Inline Empty
<meta/> Provides other data or information about the HTML document. This data is not displayed on the HTML document. N/A Empty
<nav> It defines a set of navigation links. Block Container
<ol> It an ordered list. Block Container
<p> It defines a paragraph Block Container
<script> It contains scripting statements. Inline Container
<section> It defines sections in the document, such as chapters, headers, footers, etc.. Block Container
<strong> It defines important text. Inline Container
<style> It is used to define style information on the HTML document. N/A Container
<table> Defines a table in the HTML document. Block Container
<td> Defines a standard cell in an HTML table. Block Container
<th> Defines a header cell in an HTML table. Block Container
<title> It defines the title of the document. Required in all HTML docs. Block Container
<tr> It defines a row in an HTML table. Block Container
<ul> It defines an unordered list. Represented by buleted list. Block Container
